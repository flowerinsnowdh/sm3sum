# sm3sum
模仿 sha1sum 写出的极简计算文件 [信息安全技术 SM3密码杂凑算法 (GB/T 32905——2016)](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=45B1A67F20F3BF339211C391E9278F5E) 散列值的简易程序

# 用法
```shell
sm3sum [文件1] [文件2] [文件3]...
```

需要 Java 21
